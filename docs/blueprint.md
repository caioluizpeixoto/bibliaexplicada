# **App Name**: Study Bible Hub

## Core Features:

- Content Display: Display Bible study content including text, images, and videos.
- User Authentication: Allow users to create accounts and log in to save progress.
- Progress Tracking: Track user progress and highlight content.
- Subscription Management: Allow users to subscribe to different Bible study plans.
- Search: Allow users to search the Bible.
- AI-Powered Summarization: Use AI to generate summaries of sections of text to help users learn more quickly. Uses tool to determine what to summarize and if any scriptural context needs to be added to the summaries.

## Style Guidelines:

- Primary color: Gold (#D4AF37) for elegance and spirituality.
- Background color: Dark brown (#2B2520) for sophistication.
- Accent color: Warm Gray (#8B8680) for text, not cold.
- Display font: 'Playfair' serif for main titles, elegant.
- Body font: 'PT Sans' sans-serif for body text, clean and readable.
- Use elegant, minimalist icons related to Bible study topics.
- Vertical blocks, alternating dark and light backgrounds.
- Subtle hover effects on buttons and elements.
